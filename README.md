# 📱 Social Media Analytics for Strategic Branding

## 📌 Project Overview

A business analytics project that evaluates social media performance using Microsoft Excel to uncover engagement trends, campaign effectiveness, and actionable marketing insights.

---

## 🎯 Business Problem

Marketing teams often struggle to determine:

- Which campaigns perform best?
- Which content generates the highest engagement?
- Which platforms produce the greatest ROI?
- How can future campaigns be optimised?

---

## 🛠 Technologies Used

- Microsoft Excel
- Pivot Tables
- Charts
- Data Cleaning
- Data Visualisation

---

## 🔄 Workflow

1. Data Collection
2. Data Cleaning
3. KPI Analysis
4. Dashboard Creation
5. Business Insights

---

## 📊 Results

- Identified top-performing campaigns
- Compared engagement metrics
- Improved performance tracking
- Generated marketing insights

---

## 💡 Business Recommendations

- Focus on high-engagement content
- Optimise posting schedules
- Improve campaign targeting
- Monitor KPIs continuously
